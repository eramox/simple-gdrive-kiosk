# This project goal is to provide a solution to 

## Process

It works the following way:

1. A powerpoint is edited on google drive
1. It is placed in a share folder
1. Its share link is placed in the file "to_publish"
1. The board download the file "to_publish"
1. If the content of the file changed, the board download the new document
1. The board transforms the ppt into pictures
1. The board write HTML code to show the new content
1. The HTTP server is updated with new code
1. The HTTP server publish the new code
1. The board open its browser an connect to the server
